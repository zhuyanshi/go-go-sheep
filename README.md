# Installation

Clone this repo,

```bash
git clone https://github.com/BurningLutz/go-go-sheep.git your-dir && cd your-dir
```

and then, install.(you can use `cnpm` instead of `npm`)

```bash
npm install
```

# Run Develop Server

```bash
npm start
```

# Build

```bash
npm run build
```
